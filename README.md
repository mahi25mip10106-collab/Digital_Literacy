# Digital_Literacy
This project is based on a digital education and application of it in our day to day lives.
Module 1: Hands-on Training on Presentation Software
This module focuses on building practical skills in creating effective digital presentations. It teaches how to use tools like PowerPoint or Google Slides to design visually appealing slides. Key concepts include slide layouts, themes, adding images, charts, animations, and transitions. It also emphasizes presenting information clearly, maintaining consistency, and using minimal text for better audience engagement.

Module 2: Building a Digital Portfolio
This module explains how to create and maintain a professional digital portfolio. A digital portfolio is an online collection of a student’s work, achievements, and skills. It covers selecting the right platform, organizing content, adding projects, certificates, and personal details. The module highlights the importance of showcasing skills effectively to improve career opportunities and online presence.

Module 3: Programming Platforms & Collaborative Learning Platforms
This module introduces platforms used for coding and collaborative learning. It explains how platforms like Kaggle help in learning data science, coding, and participating in competitions. It also covers collaborative tools like Google Docs, GitHub, and online forums where students can work together, share ideas, and learn from others. The focus is on teamwork, problem-solving, and continuous learning.

Module 4: Email Etiquette & Basics of Social Media Etiquette
This module teaches professional communication in the digital world. It includes proper email writing skills such as clear subject lines, polite language, formal tone, and structured content. It also explains social media etiquette like respecting others, avoiding oversharing, maintaining professionalism, and being responsible online. The goal is to build a positive digital identity.

Module 5: Awareness on Cyber Crime and Financial Fraudulent
This module creates awareness about online threats and safety. It explains types of cyber crimes such as phishing, hacking, identity theft, and online scams. Special focus is given to financial safety like UPI fraud, OTP scams, and secure transactions. It also provides preventive measures like strong passwords, avoiding suspicious links, and reporting cyber crimes through official channels like the National Cyber Crime Portal and helpline 1930.
